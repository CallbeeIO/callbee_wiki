# Поддерживаемые системы

## IP-АТС

| АТС | Подключение | Примечания |
|---|---|---|
| **FreePBX** 13+ | AMI (порт 5038) | Asterisk 13 и выше |
| **Yeastar S-серия** (S20/S50/S100/S300) | AMI + API (8088) или FTP (21) для S20 | S20 — через FTP, остальные — через API |
| **Yeastar P-серия** (P550/P560/P570) | App Center | Требуется активная лицензия App Center |

## CRM-системы

| CRM | Тип интеграции | Где установить |
|---|---|---|
| **Битрикс24** | Приложение | [Битрикс24.Маркет](https://www.bitrix24.ru/apps/?app=callbee.callbee) |
| **amoCRM** | Виджет | amoМаркет |
| **ROISTAT** | Интеграция | Каталог интеграций ROISTAT |
| **1С** 8.3.27+ | Расширение | Маркетплейс расширений 1С |

## Требования

> [!NOTE] Общие требования для всех конфигураций
> - **Статический IP-адрес** на сервере АТС (или проброс портов на роутере)
> - Доступ к TCP-портам АТС с [IP-адресов сервиса Callbee](/ip-addresses/)
> - Активная лицензия CRM-системы
> - Личный кабинет [my.callbee.io](https://my.callbee.io)
