# Поддерживаемые системы

## IP-АТС

| АТС | Версия | Подключение | Примечания |
|---|---|---|---|
| FreePBX | 13 и выше | AMI (порт 5038) | Требуется chan_sip |
| Yeastar S20 | S-серия | AMI + FTP (порт 21) | API отсутствует |
| Yeastar S50 | S-серия | AMI + API (порт 8088) | — |
| Yeastar S100 | S-серия | AMI + API (порт 8088) | — |
| Yeastar S300 | S-серия | AMI + API (порт 8088) | — |

## CRM-системы

| CRM | Тип интеграции | Маркетплейс |
|---|---|---|
| Битрикс24 | Приложение | Битрикс24 Маркетплейс |
| amoCRM | Виджет | amoМаркет |

## Требования

> [!NOTE] Общие требования для всех конфигураций
> - **Статический IP-адрес** на сервере АТС
> - Доступ к TCP-портам АТС с [IP-адресов сервиса Callbee](/ip-addresses/)
> - Активная лицензия CRM-системы
> - Личный кабинет [my.callbee.io](https://my.callbee.io)
