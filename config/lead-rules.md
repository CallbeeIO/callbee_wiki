# Правила создания лидов и сделок

Callbee может автоматически создавать сущности в CRM при поступлении звонков.

## Типы сущностей

+++ Битрикс24
- **Лид** — создаётся при звонке с неизвестного номера
- **Контакт** — создаётся автоматически или вручную
- **Сделка** — может создаваться вместе с лидом
+++ amoCRM
- **Контакт** — создаётся при звонке с неизвестного номера
- **Сделка** — может создаваться автоматически
+++

## Настройка правил

1. Войдите в [личный кабинет](https://my.callbee.io)
2. Откройте настройки сервиса
3. Перейдите в раздел **«Правила создания сущностей»**
4. Настройте условия создания лидов/сделок

> [!NOTE]
> Правила задаются на этапе создания сервиса (шаг 4) и могут быть изменены позже в настройках.
