# Сетевые настройки для Yeastar

Для подключения Callbee к вашей АТС необходим **статический IP-адрес** и проброс портов через NAT.

## Необходимые порты

| Порт | Протокол | Назначение | Модели |
|---|---|---|---|
| **5038** | TCP | AMI (Asterisk Manager Interface) | Все модели |
| **8088** | HTTP/HTTPS | API Yeastar | S50, S100, S300 |
| **21** | TCP | FTP (доступ к записям) | Только S20 |

## Настройка NAT

Пробросьте указанные порты на вашем маршрутизаторе к внутреннему IP-адресу АТС Yeastar.

> [!CAUTION] Безопасность
> Рекомендуется ограничить доступ к портам только для [IP-адресов сервиса Callbee](/ip-addresses/).
