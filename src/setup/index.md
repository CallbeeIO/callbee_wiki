# Установка Callbee

Выберите вашу IP-АТС для перехода к инструкции:

||| FreePBX 13+
[!ref](/setup/freepbx/)
АТС на базе Asterisk. Подключение через AMI.
||| Yeastar S-серия
[!ref](/setup/yeastar/)
IP-АТС Yeastar S20, S50, S100, S300. Подключение через AMI + API/FTP.
||| Yeastar P-серия
[!ref](/setup/yeastar-p/)
IP-АТС Yeastar P550, P560, P570. Подключение через App Center.
|||

> [!CAUTION] Важно
> Перед началом установки убедитесь, что ваша система соответствует [требованиям](/overview/supported-systems/).
